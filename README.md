* React

### How to run

Run locally on `http://localhost:8888`

    npm install
    npm start
    
Swagger ui  `http://localhost:8080/swagger-ui.html`

### How to build prod

    npm run build
    
### How to check style

    npm lint
    
### How to test

    npm test
